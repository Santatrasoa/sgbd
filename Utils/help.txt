COMMANDES DISPONIBLES
DATABASE
create_database <name>       : Créer une nouvelle base de données
create_db <name>             : Alias pour create_database
drop_database <name>         : Supprimer une base de données
drop_db <name>               : Alias pour drop_database
use_database <name>          : Utiliser une base de données
use_db <name>                : Alias pour use_database
leave_database               : Quitter la base de données actuelle
leave_db                     : Alias pour leave_database
list_database                : Lister toutes les bases de données
list_db                      : Alias pour list_database

TABLES
create_table <name>(col:type[constraint], ...) : Créer une table
add_into_table <table>(col=value, ...)         : Ajouter des données
drop_table <name>                             : Supprimer une table
list_table                                    : Lister les tables de la BD utilisée
describe_table <table>                        : Décrire la table et ses colonnes

UTILISATEURS
create_user <name> password=<pwd> [role=<role>] : Créer un utilisateur
list_user                                        : Lister tous les utilisateurs
drop_user <name>                                 : Supprimer un utilisateur
switch_user_to <name> password=<pwd>                  : Se connecter comme utilisateur

PERMISSIONS
grant <permission> on <table|*> to <user>      : Donner un droit
revoke <permission> on <table|*> from <user>   : Retirer un droit
show_grants <user>                              : Montrer les droits d'un utilisateur

AUTRES
clear                                           : Nettoyer l'écran
exit                                            : Quitter le SGBD
